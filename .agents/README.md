# Spectra skill maintenance

The Codex skills in this directory were generated by Spectra 2.3.1 and then patched for this repository's workflow and for verified CLI behavior.

Local fixes include:

- OpenSpec paths and plan discovery under `docs/planning/`
- clean-worktree and staged-index safety checks
- archive-time delta-spec handling and post-commit tracking cleanup
- active, Git-visible proposals instead of automatic parking
- disposable PostgreSQL requirements for API test suites

`spectra update` overwrites generated skill files, including these fixes. Before accepting an update, save the current diff, regenerate, re-audit the commands against the installed CLI, and reapply any fixes that are still necessary.
